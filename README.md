# scam-call-checker
A Python command-line tool to check and report scam phone numbers, helping users identify fraudulent calls and protect themselves from scams.

## Description

Scam Call Checker - Phone Number Verification Tool

A Python-based application that helps users identify and report scam phone numbers. 
Users can check if a number has been reported as a scam, view risk levels and 
scam categories, and contribute by reporting new fraudulent numbers to help 
protect others.

Features:
• Check phone numbers against a community database of reported scams
• Report new scam numbers with category classification
• View risk levels (Low/Moderate/High) based on report frequency
• Get safety recommendations and scam explanations
• Support for international phone number formats

Built as a CS50 Python final project to address the growing problem of phone 
scams, particularly prevalent in India where thousands of illegal call centers 
operate to defraud people worldwide.

Technologies: Python, Requests library, MockAPI, pytest
